# 🍬 Candy Crush Game

A fun and colorful **Candy Crush inspired game** built with smooth gameplay, attractive UI, and interactive animations. Match candies, score points, and enjoy an addictive puzzle experience directly in your browser/app.

---

## ✨ Features

- 🎮 Classic Candy Crush style gameplay
- 🍭 Match 3 or more candies
- ⚡ Smooth animations and effects
- 🏆 Score tracking system
- 📱 Responsive design for all devices
- 🎨 Clean and modern UI
- 🔊 Sound effects and game interactions

---

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript

## 🛠️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/candy-crush-game.git
```

2. Open the project folder

```bash
cd candy-crush-game
```

3. Run the project

- Open `index.html` in your browser

## 🎯 Gameplay

- Swap adjacent candies
- Match 3 or more same candies
- Earn points and create combos
- Beat your high score
